TStyle

  • Add the fill color attribute (SetLegendFillColor() and the font attribute (SetLegendFont().
  • A new TStyle called "Clean" has been implemented. It can be set with: 
         gROOT->SetStyle("Clean");
    It has very little decoration. It was made looking at the default styles usually used by the experiments.
    A new parameter Canvas.Style in config/rootrc.in allows to define the default style. If it is not specified, the Clean style is used. To use the old default style on can set it to Classic.
